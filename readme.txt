Create fast, responsive, and filterable blog grids with category & tag filters, pagination, and mobile off-canvas filters for Elementor.

== Description ==

**Dynamic Blog Grid & Filters for Elementor** lets you build modern, responsive, and highly customizable blog/post grids directly inside Elementor.

Designed for performance and UX, this plugin provides:

- Live category & tag filters
- Clean grid-based blog layouts
- Mobile off-canvas filter panel
- Pagination support
- Skeleton preloader
- Touch-friendly UI

Perfect for blogs, magazines, news portals, and content-heavy websites.

== Features ==

- Elementor native widget
- Filter posts by category and tag
- Responsive grid layout (desktop, tablet, mobile)
- Mobile-only off-canvas filter panel
- Search inside filters
- Filter item counters
- Pagination support
- Lightweight & performance optimized
- Developer-friendly structure

== How It Works ==

1. Install and activate Elementor
2. Install and activate this plugin
3. Open a page with Elementor
4. Drag **Dynamic Blog Grid & Filters** widget
5. Configure grid layout, filters, pagination, and styles


== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the **Plugins** menu
3. Ensure Elementor is installed and activated
4. Start building pages with Elementor

== Frequently Asked Questions ==

= Does this plugin require Elementor? =

Yes. Elementor must be installed and activated.

= Does it support custom post types? =

Not in version 1.0. Support for CPTs is planned.

= Is AJAX filtering supported? =

Yes. Posts are filtered and paginated using AJAX for a smooth user experience.

= Is this plugin free? =

Yes. A PRO version is planned with advanced features.
